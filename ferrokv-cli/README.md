# Ferrokv CLI

The command-line interface for [Ferrokv](https://github.com/xosnrdev/ferrokv).

It allows you to interact with a Ferrokv database from the terminal, which is useful for testing, debugging and introspection.

## Installation

Via cargo by running:

```sh
cargo install ferrokv-cli
```

## Usage

Run `ferrokv help` to see available commands and options.
